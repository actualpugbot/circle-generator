# Circle Generator

Static web app for planning Minecraft block circles with a clean top-down preview.
It computes outline circles in a Web Worker for fast updates as you tweak settings.

## Run Locally

No build step is required. Serve the repository root as static files, then open `index.html`.

Example:

```bash
python3 -m http.server 8080
```

Then visit `http://localhost:8080`.
